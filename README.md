# 🖋️ Half Tattoo - Landing Page

Esta é uma Landing Page profissional desenvolvida para o estúdio **Half Tattoo**, localizado em Vargem Grande, Florianópolis - SC. O projeto foi focado em performance, design moderno e conversão de clientes via WhatsApp.

## 🚀 Tecnologias Utilizadas

* **HTML5** (Semântico e estruturado)
* **Tailwind CSS** (Estilização utility-first e responsividade)
* **JavaScript** (Funcionalidades interativas e links dinâmicos)
* **Vercel / Netlify** (Hospedagem e Deploy Contínuo)

## 🎯 Diferenciais do Projeto

* **Design Dark Premium:** Estética sóbria com tons de cinza e preto para destacar os trabalhos de tatuagem.
* **Foco em Conversão:** Botão de agendamento via WhatsApp estrategicamente posicionado.
* **Mobile-First:** Totalmente otimizado para visualização em smartphones, garantindo que o cliente agende a tattoo de qualquer lugar.
* **Informativo:** Destaque para o horário diferenciado de atendimento (até às 23h).
